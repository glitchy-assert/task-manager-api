# Task Manager API

Multi-user task management system built with FastAPI.

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy 2.0 (async)
- Alembic (migrations)
- JWT Authentication

## Features (Planned)

- [ ] User authentication (JWT)
- [ ] Teams management
- [ ] Projects
- [ ] Tasks with assignments
- [ ] Comments
- [ ] File attachments
- [ ] Real-time notifications
